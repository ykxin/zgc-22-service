# AI 双向纠纷公允判定系统 — 需求文档

## 一、背景与痛点

家政服务行业纠纷处理存在以下核心问题：

- **维权难**：服务过程缺乏客观记录，举证困难
- **判定主观**：人工仲裁依赖个人经验，缺乏标准
- **偏袒单方**：平台立场不中立，双方均缺乏信任
- **双向猜忌**：雇主担心服务不达标，从业者担心恶意差评或克扣薪资

---

## 二、功能目标

搭建 AI 中立智能仲裁系统，实现纠纷处理**客观、公平、公正**，彻底化解双向猜忌，平衡雇主消费权益与从业者劳动权益。

---

## 三、功能需求

### 3.1 多维度数据溯源取证

**目标：** 服务全程自动留存不可篡改的客观凭证。

| 数据类型 | 说明 |
|----------|------|
| 录音记录 | 服务过程音频，支持语音语义分析 |
| 服务打卡记录 | 从业者按 SOP 步骤打卡，含时间戳与地理位置 |
| AI 验收报告 | 服务完成后自动生成的质量评估报告 |
| 时序服务轨迹 | 完整服务行为时间线，还原服务全过程 |

**非功能要求：**
- 所有数据自动存档，加密存储
- 数据不可篡改，具备完整性校验
- 支持在纠纷发起时一键调取关联证据

---

### 3.2 服务视频上传与 AI 自动分析

**目标：** 从业者可上传服务过程视频，AI 自动解析家务执行情况并生成综合评价，作为取证与判定的核心依据。

**上传要求：**

| 项目 | 说明 |
|------|------|
| 上传时机 | 服务进行中或服务完成后，关联至对应订单 |
| 文件格式 | MP4、MOV，单文件不超过 500MB |
| 上传数量 | 每个订单最多上传 5 段视频 |
| 存储方式 | 加密存储，与订单绑定，不可删除（纠纷期间锁定） |

**AI 视频分析内容：**

| 分析维度 | 说明 |
|----------|------|
| 家务动作识别 | 识别擦拭、拖地、整理、消毒等具体操作行为 |
| SOP 步骤覆盖率 | 比对视频中出现的操作与标准 SOP 步骤，计算覆盖比例 |
| 操作规范性评估 | 判断动作是否符合服务标准（如消毒液用量、擦拭方向等） |
| 服务时长核验 | 统计有效操作时长，排除空置、静止片段 |
| 异常行为检测 | 识别潜在的物品接触异常、区域遗漏等风险行为 |

**AI 综合评价输出：**

- **视频评分**（0-100 分）：综合动作规范性、SOP 覆盖率、服务时长加权计算
- **操作摘要**：列出识别到的主要家务步骤及完成质量
- **风险提示**：标注疑似遗漏项或异常操作片段（含时间戳）
- **与打卡记录的一致性比对**：视频内容与从业者打卡步骤是否吻合

**与纠纷仲裁的联动：**

- 纠纷发起时，系统自动将关联订单的视频分析报告纳入证据包
- 视频评分作为 `analyze_dispute()` 判定的独立维度，权重与验收得分并列
- 若视频评分与雇主评价存在显著差异（如视频评分 ≥ 80 但雇主给差评），系统自动标记为疑似恶意差评

**非功能要求：**
- 视频分析结果不可人工修改，仅平台管理员可申请复核
- 分析完成时间目标：上传后 5 分钟内返回结果
- 视频及分析报告保留至订单结束后 180 天

---

### 3.3 AI 智能责任判定

**目标：** 通过多维数据分析，精准界定纠纷责任归属。

**触发场景：**
- 工资争议
- 服务差评
- 物品损耗
- 服务质量纠纷

**判定方法：**

| 分析维度 | 具体手段 |
|----------|----------|
| 语音语义分析 | 对录音内容进行语义理解，识别沟通中的承诺、争议点 |
| 服务数据比对 | 将实际打卡记录与标准 SOP 进行逐项比对 |
| SOP 标准匹配 | 判断从业者是否存在漏项、敷衍等不达标行为 |

**责任归属分类：**

- **从业者责任**：服务不达标、漏项敷衍、未按 SOP 执行
- **雇主责任**：无端追责、恶意差评、克扣薪资

---

### 3.4 自动生成仲裁报告

**目标：** 输出标准化仲裁结果，替代人工主观判断。

**报告内容：**
- 纠纷事实还原（时间线 + 证据摘要）
- 责任认定结论（含依据说明）
- 视频分析摘要（评分、操作覆盖率、风险提示）
- 处理建议（从以下类型中选取适用项）：

| 处理类型 | 适用场景 |
|----------|----------|
| 整改要求 | 从业者服务不达标 |
| 赔付建议 | 物品损耗或服务严重失职 |
| 撤销差评 | 雇主恶意差评经核实属实 |
| 薪资补发 | 雇主无故克扣薪资 |

**非功能要求：**
- 报告格式标准化，可导出存档
- 双方均可查看报告内容
- 平台管理员可复核并执行处理建议

---

## 四、权益保障原则

| 对象 | 保障内容 |
|------|----------|
| 雇主 | 服务质量有据可查，消费权益有保障 |
| 从业者 | 劳动尊严受尊重，合法收入不被侵害 |
| 平台 | 中立立场，依据客观数据裁决，规避人工偏差 |

---
